How would you determine if there is a duplicate item in an array?

Trivial solution: \(O(n^2)\)

Input = A size n
for i = 1...n
for j = i+1...n
if A[i] == A[j]
return yes
return no

Better: \(O(n)\)

let H be a hash table for i = 1...n
if H.contains(A[i])
return yes
else H.add(A[i])
return no

Another solution: \(O(nlogn) + O(n) = O(nlogn)\)

Sort(A)
for i = 1...n
if A[i] == A[i+1]
return yes
return no

Recall asymptotic notations:
\(g(n) = O(f(n)) \leftrightarrow \lim_{n\rightarrow\infty} \frac{g(n)}{f(n)} \leq c\)
\(g(n) = o(f(n)) \leftrightarrow \lim_{n\rightarrow\infty} \frac{g(n)}{f(n)} = 0\)
\(g(n) = \Theta(f(n) \leftrightarrow g(n) = O(f(n)) & f(n) = O(g(n))\)

Given an array, find the min and max element

Trivial solution: Go through each element and compare for the max

min <- A[1], max <- A{i]
for i = 2...n
if A[i] > max then max \(\leftarrow\) A[i]
else if A[i] < min then min \(\leftarrow\) A[i]

Better: do pairs and then compare the lower and max
